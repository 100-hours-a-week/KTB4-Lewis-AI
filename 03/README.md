# 회고

우선 미니퀘스트의 양이 은근 많아 힘들었지만 모두 완료했다. Numpy, Pandas,matplotlib같은 라이브러리는 데이터 시각화, 데이터 전처리 뿐 아니라 모델의 행렬 연산 및 모델 개발과 분석에 있어서 꼭 알아야 한다 특히 NumPy는 딥러닝 프레임워크의 Tensor 구조와 맞닿아 있어서, NumPy를 익혀두면 나중에 모델을 다룰 때도 기반이 된다. 과제 내용이 이런 기술들을 단계별로 진행할수 있어서 좋았다. 사실 전공에서 어느정도 해본적이 있었기에 큰 어려움은 없었지만 오래전에 했던 내용이라 기억을 살려가며 진행했다.
